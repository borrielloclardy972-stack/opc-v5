# OPC 最终稳定上线版

这是完整 UI 展示版，不再是入口壳 Demo。

## 已包含

- 小鹅通/企业SaaS风格登录页
- 用户登录、管理员登录、注册申请
- 首页公司介绍与学习收益
- 代理认证卡密：等级、分数、开单、金额、排名
- 培训营 / 训练营三级菜单
- 产品物料资料、素材案例、其他资料
- 学习进度
- 考核学习进度（规划中）
- 管理员上传、删除、本地注册申请查看
- 无外部图片依赖，平台 Logo 用内置徽标，不会出现图片 404

## GitHub Pages 上传

解压 ZIP 后，把里面所有文件直接上传到仓库根目录。根目录必须直接看到：

```text
index.html
styles.css
app.js
404.html
.nojekyll
```

不要上传外层文件夹，不要只上传 ZIP。
